# Linear Regression

## Overview

This notebook demonstrates the implementation of **Linear Regression** for predicting a continuous target variable based on input features.

Linear Regression is one of the most fundamental algorithms in **Machine Learning** and is commonly used for predictive modeling and statistical analysis.

## Objective

The goal of this notebook is to:

* Understand the intuition behind linear regression
* Implement the algorithm using Python
* Train a model on a dataset
* Visualize the regression line and predictions
